## 머리말

MEAN Stack 기반으로 구조를 다듬으면서 Front-End에서 RequireJS를 조합한 소스입니다.  
NodeJS에서 사용된 모듈과 버전은 [package.json](package.json)을 확인하세요.

돌아가는 모습은 샘플 사이트를 참고하세요.  
오류/의견 및 피드백을 남겨주세요.
